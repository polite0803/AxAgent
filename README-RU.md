## Возможности

### Чат и модели

- **Поддержка нескольких провайдеров** — Совместим с OpenAI, Anthropic Claude, Google Gemini и всеми API, совместимыми с OpenAI
- **Управление моделями** — Получение списков удалённых моделей, настройка параметров (температура, максимальное количество токенов, Top-P и т.д.)
- **Ротация нескольких ключей** — Настройка нескольких API-ключей для каждого провайдера с автоматической ротацией для снижения нагрузки на лимиты
- **Потоковый вывод** — Рендеринг в реальном времени токен за токеном со сворачиваемыми блоками размышлений
- **Версии сообщений** — Переключение между несколькими версиями ответа на сообщение для сравнения эффектов моделей или параметров
- **Ветвление разговора** — Создание новых веток из любого узла сообщений с параллельным сравнением веток
- **Управление разговорами** — Закрепление, архивирование, отображение в хронологических группах и массовые операции
- **Сжатие разговора** — Автоматическое сжатие длинных разговоров с сохранением ключевой информации для экономии контекстного пространства
- **Одновременный ответ нескольких моделей** — Задать один и тот же вопрос сразу нескольким моделям с параллельным сравнением ответов

### AI Agent

- **Поддержка нескольких провайдеров** — Совместим с OpenAI, Anthropic Claude, Google Gemini и всеми API, совместимыми с OpenAI, с поддержкой Ollama для локальных моделей и удалённых шлюзов, таких как OpenClaw/Hermes
- **Режим Agent** — Переключитесь в режим Agent для автономного выполнения многошаговых задач: чтение/запись файлов, выполнение команд, анализ кода и многое другое
- **Три уровня разрешений** — По умолчанию (запись требует одобрения), Принять правки (автоматическое одобрение изменений файлов), Полный доступ (без запросов) — безопасно и контролируемо
- **Песочница рабочего каталога** — Операции Agent строго ограничены указанным рабочим каталогом, предотвращая несанкционированный доступ
- **Панель одобрения инструментов** — Отображение запросов вызовов инструментов в реальном времени с пошаговой проверкой, «всегда разрешать» одним щелчком или отклонить
- **Отслеживание затрат** — Статистика использования токенов и затрат в реальном времени для каждой сессии

### Мультиагентная система

- **Координация суб-агентов** — Создание нескольких суб-агентов, формирующих архитектуру координации «ведущий-ведомый»
- **Параллельное выполнение** — Обработка нескольких агентов параллельно для повышения эффективности сложных задач
- **Состязательные дебаты** — Несколько агентов обсуждают разные точки зрения для получения лучших решений через столкновение идей
- **Движок рабочих процессов** — Мощная оркестрация рабочих процессов с поддержкой условных ветвлений, циклов и параллельного выполнения
- **Командные роли** — Назначение конкретных ролей разным агентам (ревью кода, тестирование, документация и т.д.) для совместного выполнения задач

### Система навыков

- **Маркетплейс навыков** — Встроенный маркетплейс навыков для просмотра и установки навыков от сообщества
- **Создание навыков** — Создание навыков из предложений с Markdown-редактором
- **Эволюция навыков** — ИИ автоматически анализирует и улучшает существующие навыки для лучшей производительности
- **Подбор навыков** — Умные рекомендации релевантных навыков, автоматически применяемых к подходящим сценариям разговора
- **Локальная регистрация навыков** — Поддержка пользовательских локальных инструментов, зарегистрированных как навыки
- **Плагинные хуки** — Поддержка pre/post хуков для внедрения пользовательской логики до/после выполнения навыков
- **Атомные навыки** — Гранулярные компоненты навыков, поддерживающие построение сложных рабочих процессов
- **Разложение навыков** — Автоматическое разложение сложных задач на исполняемые атомные навыки
- **Сгенерированные инструменты** — ИИ автоматически генерирует и регистрирует новые инструменты для расширения возможностей агента

### Система рабочих процессов

- **Редактор рабочих процессов** — Визуальный конструктор рабочих процессов с перетаскиванием, соединением и настройкой узлов
- **Шаблоны рабочих процессов** — Встроенные пресеты для быстрого запуска распространённых задач
- **Управление версиями** — Шаблоны рабочих процессов поддерживают управление версиями с возможностью отката к историческим версиям
- **Движок рабочих процессов** — Мощный движок выполнения рабочих процессов с поддержкой параллельного, условного и цикличного выполнения
- **История выполнения** — Подробная запись истории выполнения рабочих процессов с отслеживанием состояния и отладкой
- **Помощь ИИ** — Помощь ИИ в дизайне рабочих процессов, автоматическая генерация и оптимизация

### Рендеринг контента

- **Рендеринг Markdown** — Полная поддержка подсветки кода, математических формул LaTeX, таблиц и списков задач
- **Редактор кода Monaco** — Monaco Editor встроен в блоки кода с подсветкой синтаксиса, копированием и предварительным просмотром diff
- **Рендеринг диаграмм** — Встроенный рендеринг блок-схем Mermaid и архитектурных диаграмм D2
- **Панель Artifact** — Фрагменты кода, черновики HTML, заметки Markdown и отчёты, просматриваемые в отдельной панели
- **Инспектор сессий** — Отображение структуры сессии в виде дерева в реальном времени для быстрой навигации к любому сообщению

### Поиск и знания

- **Веб-поиск** — Интеграция с Tavily, Zhipu WebSearch, Bocha и другими, с аннотациями источников цитирования
- **Локальная база знаний (RAG)** — Поддержка нескольких баз знаний; загрузите документы для автоматического разбора, разбивки на части и векторной индексации с семантическим извлечением релевантных отрывков во время разговоров
- **Граф знаний** — Граф отношений сущностей знаний, визуализирующий связи между точками знаний
- **Система памяти** — Мультипространственная память с ручным вводом или автоматическим извлечением ключевой информации с помощью ИИ
- **Полнотекстовый поиск** — Движок FTS5 для быстрого поиска по разговорам, файлам и памяти
- **Управление контекстом** — Гибкое прикрепление файловых вложений, результатов поиска, отрывков из базы знаний, записей памяти и результатов инструментов

### Инструменты и расширения

- **Протокол MCP** — Полная реализация Model Context Protocol с поддержкой транспортов stdio и HTTP/WebSocket
- **OAuth-аутентификация** — Поддержка потока аутентификации OAuth для серверов MCP
- **Встроенные инструменты** — Готовые к использованию встроенные инструменты для файловых операций, выполнения кода, поиска и многого другого
- **Панель выполнения инструментов** — Визуальное отображение запросов вызовов инструментов и возвращённых результатов
- **LSP-клиент** — Встроенная поддержка протокола LSP для интеллектуального автодополнения кода и диагностики

### API-шлюз

- **Локальный API-шлюз** — Встроенный локальный API-сервер с нативной поддержкой интерфейсов, совместимых с OpenAI, Claude и Gemini
- **Внешние ссылки** — Интеграция в один клик с внешними инструментами, такими как Claude CLI и OpenCode, с автоматической синхронизацией API-ключей
- **Управление API-ключами** — Создание, отзыв и включение/отключение ключей доступа с описательными заметками
- **Аналитика использования** — Анализ объёма запросов и использования токенов по ключу, провайдеру и дате
- **Диагностические инструменты** — Проверки здоровья шлюза, тесты подключения и отладка запросов
- **Поддержка SSL/TLS** — Встроенная генерация самоподписанных сертификатов с поддержкой пользовательских сертификатов
- **Журналы запросов** — Полная запись всех API-запросов и ответов, проходящих через шлюз
- **Шаблоны конфигурации** — Готовые шаблоны интеграции для популярных CLI-инструментов, таких как Claude, Codex, OpenCode и Gemini
- **Коммуникация в реальном времени** — Push-событий WebSocket в реальном времени, совместимый с OpenAI Realtime API

### Данные и безопасность

- **Шифрование AES-256** — API-ключи и конфиденциальные данные шифруются локально с помощью AES-256-GCM
- **Изолированные каталоги данных** — Состояние приложения в `~/.axagent/`; пользовательские файлы в `~/Documents/axagent/`
- **Автоматическое резервное копирование** — Плановое автоматическое резервное копирование в локальные каталоги или хранилище WebDAV
- **Восстановление из резервной копии** — Восстановление в один клик из исторических резервных копий
- **Экспорт разговоров** — Экспорт разговоров в виде PNG-скриншотов, Markdown, обычного текста или JSON
- **Управление пространством хранения** — Визуальное отображение использования диска с очисткой ненужных файлов

- **Переключение тем** — Тёмные/светлые темы, следующие системным предпочтениям или устанавливаемые вручную
- **Язык интерфейса** — Полная поддержка упрощённого китайского, традиционного китайского, английского, японского, корейского, французского, немецкого, испанского, русского, хинди и арабского языков

- **Всегда поверх** — Закрепление главного окна поверх всех остальных окон
- **Глобальные горячие клавиши** — Настраиваемые глобальные сочетания клавиш для вызова главного окна в любое время
- **Автозапуск** — Необязательный запуск при старте системы

```bash

git clone https://github.com/polite0803/AxAgent.git

# Запустить в режиме разработки
```

## Часто задаваемые вопросы

### macOS: «Приложение повреждено» или «Не удаётся проверить разработчика»

Поскольку приложение не подписано Apple, macOS может показать одно из следующих сообщений:

- «AxAgent» повреждён и не может быть открыт
- «AxAgent» не может быть открыт, поскольку Apple не может проверить его на наличие вредоносного программного обеспечения

**Шаги для решения:**

**1. Разрешить приложения из «Любого источника»**

```bash
sudo spctl --master-disable
```

Затем перейдите в **Системные настройки → Конфиденциальность и безопасность → Безопасность** и выберите **Любой источник**.

**2. Удалить атрибут карантина**

```bash
sudo xattr -dr com.apple.quarantine /Applications/AxAgent.app
```

> Совет: Вы можете перетащить значок приложения в терминал после ввода `sudo xattr -dr com.apple.quarantine `.

**3. Дополнительный шаг для macOS Ventura и более поздних версий**

После выполнения вышеуказанных шагов первый запуск всё ещё может быть заблокирован. Перейдите в **Системные настройки → Конфиденциальность и безопасность** и нажмите **Всё равно открыть** в разделе «Безопасность». Это нужно сделать только один раз.
