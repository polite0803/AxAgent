## विशेषताएँ

### चैट और मॉडल

- **मल्टी-प्रोवाइडर समर्थन** — OpenAI, Anthropic Claude, Google Gemini और सभी OpenAI-संगत APIs के साथ संगत; Ollama लोकल मॉडल, OpenClaw/Hermes रिमोट गेटवे कनेक्शन के लिए भी समर्थन
- **मॉडल प्रबंधन** — रिमोट मॉडल सूचियाँ लाएँ, पैरामीटर कस्टमाइज़ करें (तापमान, अधिकतम टोकन, Top-P, आदि)
- **मल्टी-की रोटेशन** — प्रत्येक प्रोवाइडर के लिए कई API कुंजियाँ कॉन्फ़िगर करें, दर सीमा दबाव वितरित करने के लिए स्वचालित रोटेशन के साथ
- **स्ट्रीमिंग आउटपुट** — फोल्ड करने योग्य थिंकिंग ब्लॉक के साथ रियल-टाइम टोकन-दर-टोकन रेंडरिंग
- **संदेश संस्करण** — मॉडल या पैरामीटर प्रभावों की तुलना करने के लिए प्रत्येक संदेश पर कई प्रतिक्रिया संस्करणों के बीच स्विच करें
- **वार्तालाप शाखाएँ** — किसी भी संदेश नोड से नई शाखाएँ बनाएँ, शाखाओं की साइड-बाय-साइड तुलना के साथ
- **वार्तालाप प्रबंधन** — पिन करें, संग्रहीत करें, समय-समूहीकृत प्रदर्शन और बल्क ऑपरेशन
- **वार्तालाप संपीड़न** — संदर्भ स्थान बचाने के लिए मुख्य जानकारी संरक्षित करते हुए लंबी बातचीत को स्वचालित रूप से संपीड़ित करें
- **मल्टी-मॉडल एक साथ प्रतिक्रिया** — एक साथ कई मॉडलों से एक ही प्रश्न पूछें, उत्तरों की साइड-बाय-साइड तुलना के साथ
- **श्रेणी प्रणाली** — विषय-आधारित संगठन के साथ कस्टम बातचीत श्रेणियाँ

### AI Agent

- **Agent मोड** — Agent मोड में स्विच करें, AI स्वायत्त रूप से बहु-चरणीय कार्य निष्पादित कर सकता है: फ़ाइलें पढ़ना/लिखना, कमांड चलाना, कोड विश्लेषण, और अधिक
- **तीन अनुमति स्तर** — डिफ़ॉल्ट (लिखने के लिए अनुमोदन आवश्यक), संपादन स्वीकार करें (फ़ाइल परिवर्तन स्वचालित रूप से स्वीकृत), पूर्ण पहुँच (कोई संकेत नहीं) — सुरक्षित और नियंत्रणीय
- **कार्य निर्देशिका सैंडबॉक्स** — Agent संचालन निर्दिष्ट कार्य निर्देशिका तक सख्ती से सीमित, अनधिकृत पहुँच को रोकता है
- **टूल अनुमोदन पैनल** — टूल कॉल अनुरोधों का रीयल-टाइम प्रदर्शन, प्रति-टूल समीक्षा, एक-क्लिक "हमेशा अनुमति दें", या अस्वीकार
- **लागत ट्रैकिंग** — प्रति सत्र रीयल-टाइम टोकन उपयोग और लागत सांख्यिकी
- **रुकें/जारी रखें** — किसी भी समय Agent कार्यों को रोकें, समीक्षा के बाद जारी रखें
- **Bash कमांड निष्पादन** — स्वचालित जोखिम सत्यापन के साथ सैंडबॉक्स वातावरण में Shell कमांड निष्पादित करें

### मल्टी-एजेंट सिस्टम

- **सब-Agent समन्वय** — मास्टर-स्लेव समन्वय आर्किटेक्चर के साथ कई सब-Agent बनाएँ
- **समानांतर निष्पादन** — जटिल कार्यों पर बेहतर दक्षता के लिए कई Agent को समानांतर रूप से संसाधित करें
- **विखंडन बहस** — विचारों की टक्कर से बेहतर समाधान उत्पन्न करने के लिए कई Agent विभिन्न दृष्टिकोणों पर बहस करते हैं
- **वर्कफ़्लो इंजन** — सशर्त शाखाओं, लूप और समानांतर निष्पादन का समर्थन करने वाला शक्तिशाली वर्कफ़्लो ऑर्केस्ट्रेशन
- **टीम भूमिकाएँ** — सहयोगी कार्य पूर्ण करने के लिए विभिन्न Agent को विशिष्ट भूमिकाएँ (कोड समीक्षा, परीक्षण, प्रलेखन, आदि) सौंपें

### स्किल सिस्टम

- **स्किल मार्केटप्लेस** — समुदाय द्वारा योगदान किए गए स्किल्स को ब्राउज़ और इंस्टॉल करने के लिए इनबिल्ट मार्केटप्लेस
- **स्किल निर्माण** — प्रस्तावों से स्वचालित रूप से स्किल बनाएँ, Markdown संपादक समर्थन
- **स्किल विकास** — AI स्वचालित रूप से मौजूदा स्किल्स का विश्लेषण और सुधार करता है
- **स्किल मिलान** — उपयुक्त बातचीत परिदृश्यों पर स्वचालित रूप से प्रासंगिक स्किल्स लागू करने के लिए बुद्धिमान सिफारिशें
- **लोकल स्किल रजिस्ट्री** — पुनः उपयोग के लिए कस्टम लोकल टूल्स को स्किल्स के रूप में पंजीकृत करें
- **प्लगइन हुक** — स्किल निष्पादन से पहले और बाद में कस्टम लॉजिक इंजेक्ट करने के लिए pre/post हुक समर्थन

### सामग्री रेंडरिंग

- **Markdown रेंडरिंग** — कोड हाइलाइटिंग, LaTeX गणित सूत्र, तालिकाओं और कार्य सूचियों के लिए पूर्ण समर्थन
- **Monaco कोड एडिटर** — कोड ब्लॉक में Monaco Editor एम्बेड किया गया, सिंटैक्स हाइलाइटिंग, कॉपी और diff प्रीव्यू के साथ
- **आरेख रेंडरिंग** — Mermaid फ्लोचार्ट और D2 आर्किटेक्चर आरेख रेंडरिंग इनबिल्ट
- **Artifact पैनल** — कोड स्निपेट, HTML ड्राफ्ट, Markdown नोट्स और रिपोर्ट एक समर्पित पैनल में देखें
- **सत्र इंस्पेक्टर** — किसी भी संदेश पर त्वरित नेविगेशन के लिए ट्री व्यू में सत्र संरचना का रियल-टाइम प्रदर्शन

### खोज और ज्ञान

- **वेब खोज** — Tavily, Zhipu WebSearch, Bocha और अधिक के साथ एकीकृत, उद्धरण स्रोत एनोटेशन के साथ
- **स्थानीय ज्ञान आधार (RAG)** — कई ज्ञान आधारों का समर्थन करता है; दस्तावेज़ अपलोड करें, ऑटो पार्सिंग, चंकिंग और वेक्टर इंडेक्सिंग के लिए, बातचीत के दौरान प्रासंगिक अनुच्छेदों की सेमांटिक पुनर्प्राप्ति
- **ज्ञान ग्राफ** — ज्ञान बिंदुओं के बीच कनेक्शन को विज़ुअलाइज़ करने वाले ज्ञान इकाई संबंध ग्राफ
- **स्मृति प्रणाली** — मल्टी-नेमस्पेस मेमोरी, मैन्युअल एंट्री या AI-संचालित स्वचालित मुख्य जानकारी निष्कर्षण
- **पूर्ण-पाठ खोज** — FTS5 इंजन बातचीत, फ़ाइलों और स्मृति में त्वरित खोज के लिए
- **संदर्भ प्रबंधन** — फ़ाइल अनुलग्नक, खोज परिणाम, ज्ञान आधार अनुच्छेद, स्मृति प्रविष्टियाँ और टूल आउटपुट को लचीले ढंग से संलग्न करें

### टूल और एक्सटेंशन

- **MCP प्रोटोकॉल** — stdio और HTTP/WebSocket दोनों ट्रांसपोर्ट का समर्थन करने वाला पूर्ण Model Context Protocol कार्यान्वयन
- **OAuth प्रमाणीकरण** — MCP सर्वर के लिए OAuth प्रमाणीकरण प्रवाह समर्थन
- **इनबिल्ट टूल** — फ़ाइल ऑपरेशन, कोड निष्पादन, खोज और अधिक के लिए तुरंत उपयोग के लिए तैयार इनबिल्ट टूल
- **टूल एग्जीक्यूशन पैनल** — टूल कॉल अनुरोधों और वापसी परिणामों का दृश्य प्रदर्शन
- **LSP क्लाइंट** — बुद्धिमान कोड पूर्णता और निदान के लिए इनबिल्ट LSP प्रोटोकॉल समर्थन

### API गेटवे

- **स्थानीय API गेटवे** — OpenAI-संगत, Claude और Gemini इंटरफेस के लिए नेटिव समर्थन के साथ इनबिल्ट स्थानीय API सर्वर
- **बाहरी लिंक** — Claude CLI, OpenCode आदि जैसे बाहरी टूल्स के साथ वन-क्लिक एकीकरण, ऑटो API कुंजी सिंक
- **API कुंजी प्रबंधन** — विवरण नोट के साथ एक्सेस कुंजियाँ उत्पन्न करें, रद्द करें और सक्षम/अक्षम करें
- **उपयोग विश्लेषण** — कुंजी, प्रोवाइडर और तारीख द्वारा अनुरोध वॉल्यूम और टोकन उपयोग विश्लेषण
- **डायग्नोस्टिक टूल्स** — गेटवे स्वास्थ्य जाँच, कनेक्शन परीक्षण और अनुरोध डिबगिंग
- **SSL/TLS समर्थन** — इनबिल्ट सेल्फ-साइन्ड सर्टिफिकेट जनरेशन, कस्टम सर्टिफिकेट के समर्थन के साथ
- **अनुरोध लॉग** — गेटवे से गुजरने वाले सभी API अनुरोधों और प्रतिक्रियाओं की पूर्ण रिकॉर्डिंग
- **कॉन्फ़िगरेशन टेम्प्लेट** — Claude, Codex, OpenCode और Gemini जैसे लोकप्रिय CLI टूल के लिए प्री-बिल्ट एकीकरण टेम्प्लेट
- **रीयल-टाइम संचार** — WebSocket रियल-टाइम इवेंट पुश, OpenAI Realtime API के साथ संगत

### डेटा और सुरक्षा

- **AES-256 एन्क्रिप्शन** — API कुंजियाँ और संवेदनशील डेटा AES-256-GCM के साथ स्थानीय रूप से एन्क्रिप्ट किए गए
- **अलग डेटा डायरेक्टरी** — `~/.axagent/` में एप्लिकेशन स्थिति; `~/Documents/axagent/` में उपयोगकर्ता फाइलें
- **ऑटो बैकअप** — स्थानीय डायरेक्टरी या WebDAV स्टोरेज में अनुसूचित स्वचालित बैकअप
- **बैकअप पुनर्स्थापना** — ऐतिहासिक बैकअप से वन-क्लिक पुनर्स्थापना
- **बातचीत निर्यात** — बातचीत को PNG स्क्रीनशॉट, Markdown, सादे पाठ या JSON के रूप में निर्यात करें
- **स्टोरेज स्पेस प्रबंधन** — अनावश्यक फ़ाइलों की सफाई के साथ डिस्क उपयोग का विज़ुअल प्रदर्शन

### डेस्कटॉप अनुभव

- **थीम स्विचिंग** — डार्क/लाइट थीम जो सिस्टम प्राथमिकता का पालन करते हैं या मैन्युअल रूप से सेट किए जा सकते हैं
- **इंटरफेस भाषा** — सरलीकृत चीनी, पारंपरिक चीनी, अंग्रेजी, जापानी, कोरियाई, फ्रेंच, जर्मन, स्पेनिश, रूसी, हिंदी और अरबी के लिए पूर्ण समर्थन
- **सिस्टम ट्रे** — बैकग्राउंड सेवाओं को बाधित किए बिना विंडो बंद होने पर सिस्टम ट्रे में छोटा करें
- **हमेशा ऊपर** — मुख्य विंडो को सभी अन्य विंडो के ऊपर रखें
- **ग्लोबल शॉर्टकट** — किसी भी समय मुख्य विंडो को बुलाने के लिए कस्टमाइज़ेबल ग्लोबल कीबोर्ड शॉर्टकट
- **ऑटो स्टार्ट** — सिस्टम स्टार्टअप पर वैकल्पिक लॉन्च
- **प्रॉक्सी समर्थन** — HTTP और SOCKS5 प्रॉक्सी कॉन्फ़िगरेशन
- **ऑटो अपडेट** — स्टार्टअप पर नए संस्करणों की स्वचालित जाँच और अपडेट के लिए संकेत
- **कमांड पैलेट** — `Cmd/Ctrl+K` से सभी कमांड और सेटिंग्स पर त्वरित पहुँच

- [Rust](https://www.rust-lang.org/) 1.75+
- [npm](https://www.npmjs.com/) 10+
- Windows के लिए [Visual Studio Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/) और [Rust MSVC targets](https://doc.rust-lang.org/cargo/reference/config.html#cfgtarget) आवश्यक है

### बिल्ड चरण

```bash
# रिपॉजिटरी क्लोन करें
git clone https://github.com/polite0803/AxAgent.git
cd AxAgent

# निर्भरताएँ इंस्टॉल करें
npm install

# डेवलपमेंट मोड में चलाएँ
npm run tauri dev

# केवल फ्रंटएंड बिल्ड करें
npm run build

# डेस्कटॉप एप्लिकेशन बिल्ड करें
npm run tauri build
```

बिल्ड आर्टिफैक्ट `src-tauri/target/release/` डायरेक्टरी में हैं।

## अक्सर पूछे जाने वाले प्रश्न

### macOS: "ऐप क्षतिग्रस्त है" या "डेवलपर को सत्यापित नहीं किया जा सकता"

चूँकि एप्लिकेशन Apple द्वारा साइन नहीं किया गया है, macOS निम्नलिखित में से एक संकेत दिखा सकता है:

- "AxAgent" क्षतिग्रस्त है और इसे नहीं खोला जा सकता
- "AxAgent" को नहीं खोला जा सकता क्योंकि Apple इसे दुर्भावनापूर्ण सॉफ़्टवेयर के लिए जाँच नहीं कर सकता

**समाधान के चरण:**

**1. "कहीं से भी" ऐप्स की अनुमति दें**

```bash
sudo spctl --master-disable
```

फिर **सिस्टम सेटिंग्स → गोपनीयता और सुरक्षा → सुरक्षा** पर जाएँ और **कहीं से भी** चुनें।

**2. क्वारंटाइन विशेषता हटाएँ**

```bash
sudo xattr -dr com.apple.quarantine /Applications/AxAgent.app
```

> टिप: आप `sudo xattr -dr com.apple.quarantine ` टाइप करने के बाद टर्मिनल पर ऐप आइकन खींच सकते हैं।

**3. macOS Ventura और बाद के संस्करणों के लिए अतिरिक्त चरण**

उपरोक्त चरण पूरे करने के बाद, पहला लॉन्च अभी भी ब्लॉक हो सकता है। **सिस्टम सेटिंग्स → गोपनीयता और सुरक्षा** पर जाएँ, फिर सुरक्षा अनुभाग में **फिर भी खोलें** पर क्लिक करें। यह केवल एक बार किया जाना चाहिए।
