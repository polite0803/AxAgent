# MCP सर्वर

## MCP क्या है?

[Model Context Protocol (MCP)](https://modelcontextprotocol.io/) एक ओपन स्टैंडर्ड है जो AI मॉडलों को बाहरी टूल्स और डेटा स्रोतों के साथ इंटरैक्ट करने देता है। AxAgent MCP क्लाइंट के रूप में काम करता है — आप MCP सर्वर जोड़ते हैं, और AI वार्तालाप के दौरान उनके एक्सपोज़ टूल कॉल कर सकता है।

---

## ट्रांसपोर्ट प्रोटोकॉल

| प्रोटोकॉल | कनेक्शन | उपयोग केस | कॉन्फ़िगरेशन |
|---------|---------|---------|------------|
| **Stdio** | लोकल प्रोसेस | `npx`, `uvx`, `python` आदि से लॉन्च टूल्स | `command` + `args` + वैकल्पिक `env` |
| **SSE** | रिमोट सर्वर | रिमोट मशीन पर होस्ट Server-Sent Events एंडपॉइंट | `url` |
| **StreamableHTTP** | रिमोट सर्वर | HTTP स्ट्रीमिंग एंडपॉइंट | `url` |

---

## MCP सर्वर जोड़ना

### फॉर्म से बनाना

1. **Settings → MCP Servers** में जाएं।
2. **Add MCP Server** क्लिक करें।
3. एक नाम दर्ज करें और ट्रांसपोर्ट प्रोटोकॉल चुनें।
4. अपने चुने हुए प्रोटोकॉल के लिए फ़ील्ड भरें।
5. **Save** क्लिक करें।

### JSON इम्पोर्ट

**JSON Import** क्लिक करें और कॉन्फ़िगरेशन ऑब्जेक्ट पेस्ट करें:

```json
{
  "mcpServers": {
    "fetch": {
      "command": "uvx",
      "args": ["mcp-server-fetch"]
    }
  }
}
```
