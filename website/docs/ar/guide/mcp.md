# خوادم MCP

## ما هو MCP؟

[بروتوكول Model Context Protocol (MCP)](https://modelcontextprotocol.io/) معيار مفتوح يتيح لنماذج AI التفاعل مع الأدوات ومصادر البيانات الخارجية. يعمل AxAgent كعميل MCP — تضيف خوادم MCP ويمكن لـ AI استدعاء الأدوات التي تكشفها خلال المحادثة.

---

## بروتوكولات النقل

| البروتوكول | الاتصال | حالة الاستخدام | التكوين |
|-----------|---------|--------------|---------|
| **Stdio** | عملية محلية | أدوات مثبّتة على جهازك، يتم إطلاقها عبر `npx` أو `uvx` أو `python`، إلخ | `command` + `args` + `env` اختياري |
| **SSE** | خادم بعيد | طرف Server-Sent Events مستضاف على جهاز بعيد | `url` |
| **StreamableHTTP** | خادم بعيد | طرف HTTP للبث، بديل أحدث لـ SSE | `url` |

---

## إضافة خوادم MCP

### الإنشاء عبر النموذج

1. اذهب إلى **الإعدادات ← خوادم MCP**.
2. انقر **إضافة خادم MCP**.
3. أدخل اسماً واختر بروتوكول النقل.
4. امأ الحقول لبروتوكولك المختار.
5. انقر **حفظ**.

### استيراد JSON

انقر **استيراد JSON** والصق كائن التكوين:

```json
{
  "mcpServers": {
    "fetch": {
      "command": "uvx",
      "args": ["mcp-server-fetch"]
    }
  }
}
```
